### [二叉树的中序遍历](https://leetcode.cn/problems/binary-tree-inorder-traversal/)
给定一个二叉树的根节点 root ，返回 它的 中序 遍历 。
### 例子
![](https://assets.leetcode.com/uploads/2020/09/15/inorder_1.jpg)
```text
输入：root = [1,null,2,3]
输出：[1,3,2]
```
```text
输入：root = []
输出：[]
```
```text
输入：root = []
输出：[]
```
### 解法
迭代,不要使用递归,递归面试不给过的。
